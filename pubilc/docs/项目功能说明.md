## 1. 项目概述

**项目名称**: ELEVEN Blog Tuner  
**项目定位**: 个人博客 AI 智能助手  
**核心价值**: 根据个人笔记学习写作风格，生成具有个人文笔特色的博客文章  
**目标用户**: 博主、内容创作者、知识工作者  
**技术栈**: Python + FastAPI + Vue3 + PostgreSQL + ChromaDB

#### 3.4.2 混合模式分析

风格学习支持三种分析模式的组合：

| 分析模式 | 说明 | 技术实现 |
|---------|------|----------|
| 规则统计 (statistical) | 基于规则的统计特征提取 | `rag/style_learner.py` |
| LLM语义 (semantic) | 使用LLM进行语义分析 | `rag/style_learner.py` |
| 向量嵌入 (embedding) | 生成风格向量表示 | `rag/embedding.py` |

#### 3.4.3 风格特征维度

**统计特征**:
- vocabulary_diversity: 词汇多样性
- average_word_length: 平均词长
- unique_words_ratio: 独特词汇比例
- average_sentence_length: 平均句长
- sentence_complexity: 句子复杂度
- punctuation_density: 标点密度
- paragraph_average_length: 平均段落长度
- transition_words_ratio: 过渡词比例
- passive_voice_ratio: 被动语态比例
- first_person_ratio: 第一人称比例
- emoji_usage: 表情符号使用

**语义特征**:
- language_style: 语言风格
- tone: 语气语调
- vocabulary_level: 词汇水平
- sentence_rhythm: 句子节奏
- rhetoric_devices: 修辞手法
- emotional_tendency: 情感倾向
- perspective: 视角人称
- logic_structure: 逻辑结构
- unique_habits: 独特习惯
- target_audience: 目标受众
- domain_characteristics: 领域特征
- writing_quirks: 写作怪癖

#### 3.4.4 技术要点

- **核心实现**: `rag/style_manager.py`, `rag/style_learner.py`
- **存储方式**: JSON文件存储 (`./data/styles/`)
- **技能注册**: 自动注册为技能供Agent使用
- **增量学习**: 支持多样本融合更新风格

### 5.2 文档处理流程

```
原始文档
    ↓
Document Washer (清洗)
    - 去除多余空白
    - 标准化换行
    - 去除特殊字符
    - 支持 Markdown/TXT/PDF
    ↓
Chunker (分块)
    - 语义分块（基于段落/句子边界）
    - 递归分块
    - 固定长度分块
    - 块级元数据
    ↓
Embedding Service (向量化)
    - OpenAI Embedding
    - 本地 Ollama 模型
    - 批量向量化
    ↓
Vector DB (ChromaDB存储)
```

### 5.3 检索流程

```
查询文本
    ↓
Embedding (向量化)
    ↓
Searcher (向量检索 Top-K*2)
    ↓
Reranker (精排 Top-K)
    - Cross-encoder 精排
    - 降级简单排序
    ↓
返回结果
```


**2. 使用笔记整合模式生成文章**:
```bash
POST /api/v1/articles/generate-v2
{
  "topic": "人工智能的发展",
  "mode": "note_integration",
  "style_name": "tech_blog_style",  // 可选
  "retrieval_strategy": "hybrid",   // vector/keyword/hybrid

  "target_length": 1500
}
```

```bash
POST /api/v1/articles/generate-v2

  "mode": "note_integration",
  "note_ids": ["note-uuid-1", "note-uuid-2"],
  "style_name": "tech_blog_style",
  "target_length": 1500
}
```

**4. 使用风格+主题模式**:
