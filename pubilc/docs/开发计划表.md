## 风险与应对

| 风险            | 影响     | 应对措施           |
| ------------- | ------ | -------------- |
| LLM API 调用成本高 | 预算超支   | 实现本地模型备选、缓存机制  |
| Agent 协作效果不佳  | 文章质量差  | 增加人工干预环节、迭代优化  |
| 开发进度延迟        | 里程碑延期  | 每周末检查进度、及时调整计划 |
| 第三方平台 API 变更  | 发布功能失效 | 抽象发布接口、快速适配    |
